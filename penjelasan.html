# Spesifikasi Program platform: website bahasa: php & javascript framework: laravel beberapa point penting yang harus diketahui pada struktur project laravel 1. app/Http/Controllers memproses perintah seperti create, read, update, & delete 2. app/Models menghubungkan database antara controller dengan views 3. routes/web mengatur route untuk memungkinkan berpindah halaman 4. .env (environment) setting atau konfigurasi database 5. public/... menyimpan segala jenis library, gambar, icon, dsb 6. resources/views tempat dimana semua tampilan atau design dibuat, dan dipanggil
